# String

### 패턴 매칭 문제

- 텍스트에서 패턴이 존재하는 모든 위치를 찾는 문제

- 텍스트는 **t** 로 패터는 **p** 로 표현을 한다.

- 텍스트의 인덱스는 **i** 로 쓰고, 패턴은 **j** 로 표현

- 텍스트의 길이는 **n** 으로 쓰고, 패턴은 **m** 으로 표현
  
  - 비교시>  `p[j] == t[i]`



#### 패턴 매칭 알고리즘의 중요 포인트

- 일치하는 경우 (`p[j] == t[i]`)
  
  - i, j를 증가시켜서 다음 문자를 비교한다.

- 불일치하는 경우 (`p[j] != t[i]`)
  
  - 다음 매칭의 시작위치로 i, j 값을 설정
